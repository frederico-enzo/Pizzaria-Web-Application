## Requisitos para Desenvolvimento

- Clone deste repositório
- JDK 20
- PostgreSQL 15
- Build com **Maven**
- Prática de Trunk-Based com GIT

Toda a configuração do Banco de Dados é feita no arquivo `application.properties`
